HERO.IO 
Application details:
A lightweight directory of productive web and mobile apps built with React + Vite. It features a polished Hero section with gradient text and the Cabin font, a responsive All Apps grid, individual app detail pages, and simple install/installed flows with toast notifications. Styled with Tailwind/DaisyUI and designed for fast local development and easy extension.
